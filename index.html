About

What is it?

This page contains a series of small games, some are work in progress, others are simple products. The website itself acts as a cumulative project to show the knowledge I have aquired during my first quarter at ACS.

Why?

From playing to making, games have been a passion of mine from the very beginning. With the tools I've aquired from the ACS program, I've been able to explore new game programming concepts and make functional prototypes to play and enjoy all while refining my coding skills.

Why Phaser and Pygame?

I started with the Pygame API for its rendering and math capabilities allowing me to create some great game bases. With time though, the lack of modern support for the API, substantial boiler plate, and difficulty to share drove me to switch. Although not as quick to code, the Phaser API for Javascript possesses incredible functionaliy, more resources, and ridiculously easy shareablilty.

All source code can be found on my Github

Instructions

Setting Up Pygame

  1. Ensure your device has python installed
  2. Download and store the Github Repo of your desired game
  3. Navigate to the folder the game is stored in with the command line
  4. Create and activate a virtual environment
  5. Use "pip install pygame" to activate pygame
  6. Run the python file with your perfered editor
  7. Now everything should be up and running!

Setting Up Phaser

  1. Click your favorite game
  2. Enjoy!
